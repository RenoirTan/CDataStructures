◆ cds_int_log2()

CDS_INLINE int32_t cds_int_log2 ( uint64_t  n)

An extremely fast implementation of a logarithm calculator for unsigned integers. This is basically binary searching the most significant digit (i.e. 1 in base 2) by bitshifting the input number whenever required. Using unrolled loops, this function should be blindingly quick. The big-O notation for this function is O(1) as it passes through a predetermined 6 loops.

This was taken from https://stackoverflow.com/a/24748637.

See also
cds_int_log10
Parameters
nThe input number
Returns
int32_t The index of the most significant digit (i.e. the floor of the log of n). -1 if n is invalid (e.g. 0).
